MySite
一个简单的网站项目，包含了基本的服务器设置和静态文件支持。这个项目通过 Python 编写，并提供了一些日志记录功能。
项目结构
mysite/
│
├── .git                   # Git 配置文件
├── .gitattributes         # Git 属性文件
├── admin.log              # 管理员日志文件
├── audit.log              # 审计日志文件
├── logs                   # 日志目录
├── routes.py              # 路由定义文件
├── server.py              # 服务器启动脚本
├── start.bat              # Windows 启动批处理文件
├── static                 # 静态文件目录
├── welcome.py             # 欢迎页脚本
└── __pycache__            # Python 字节码缓存
